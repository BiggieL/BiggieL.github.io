Apps

Android applications that I have worked on for projects.

  • QuickDraw

    QuickDraw is a drawing game that our group created for a project, while using Kotlin for the first time to develop an android app. In QuickDraw, you are given a randomized prompt every single day and you have a small time frame to draw said prompt. Upon submitting your drawing, you can view every other users' drawings for that prompt, and vote on which one is the best! This app includes social customization, with custom profiles and the ability to add friends. I also created a backend server that is ready to deploy at any time, which means this app can be ran by anyone from any device as long as the server is up. There are currently no plans to finish this app and upload it to the appstore, but it was quite a great learning experience.

Current Projects

These are the projects I am currently working on, with group members for Senior Capstone and Reverse Engineering.

  • Pretty Good Emulator

    We aim to make Pretty Good Emulator a high quality and fluid emulation frontend, with an emphasis on a slick user experience and social features. These features include customizable profiles, statuses, the ability to add friends and play with them, and achievmenet tracking. Emulation frontends tend to be designed by programmers, for programmers, and lack a polished user experience. We aim to solve that issue with this project.

  • Metal Gear Solid 1 Decompilation

    Along with an entire community of coders, we aim to completely reverse engineer Metal Gear Solid Integral for PlayStation back to C source code, which is a monumental task. The game is split up into many different overlays which need to be decompiled seperately, which is proving to be quite the undertaking.
